Task Management System

This Task Management System is specifically designed for database teams to monitor task assignments and progress efficiently. It allows tracking of who assigned tasks to specific resources and their completion status for procedures and tables. Users can identify procedure-related tables and indexes, find details about who created an index, and check current tasks with priorities. The application includes robust features like session management, authentication, and authorization. Built using ASP.NET Core MVC, it boasts a responsive design for seamless usability across devices.

Dynamic Form Creator

This project enables users to dynamically design and generate input forms with various field types directly from the application interface. Alongside form creation, corresponding database tables are automatically generated to store the entered data. Users can view, update, or delete the saved records through a user-friendly interface. It leverages ADO.NET for database interactions, providing efficient CRUD functionality. The project emphasizes flexibility and usability for real-time form management without requiring technical expertise. This solution is ideal for businesses seeking dynamic data collection and management.
